Intro to Javascript

Will Crichton & James Yan

Description

The Internet is a powerful and constantly evolving tool for communicating and distributing applications on a massive scale. For better or for worse, the only way we can make our web pages interactive is with JavaScript, the new language du jour. JavaScript is a quirky language used in a growing number of places beyond just the browser. In this workshop, you'll learn the basics of JavaScript as well as popular JS frameworks commonly used in web development.

Times

11:00am, 3:00pm

Prerequisites

Some programming experience (familiarity with variables, types, etc.)
