Intro to Open Source

Varun

Description

Some of the most popular and most pervasive software today, for example about 99% of the web or a major part of every operating system, is made up of amazingly synchronously working open source software blocks. From Linux, and Apache, to Python, Android, jQuery, Bootstrap - you name it. But why is it some open source software is more successful and famous than others? And what does it take for creative hackers from CMU to produce open source software hacks that become as famous and awesome?

CrashCourse's workshop “Intro to Open Source Development” is where you will find the answers. From code quality to testing to licensing to publicizing to building a community, you will learn the “soft skills” that make good hacks into great open source software and communities. During this 2-part workshop, you will write a small piece of code, and learning from lessons from popular open source software, you will “correctly” turn it an open source software that the world will be able to use and enjoy and deploy.

This will make you:

  1. A rock star in the open source world
  2. A much better thinker when it comes to programming in teams
  3. Highly desireable for an internship or full-time job!

The best hack-turned-open source software will get a special prize!

Times

3:00pm (2-part, total 3 hours)

Prerequisites

  • Some programming experience
  • Laptop
  • An “open” mind!
